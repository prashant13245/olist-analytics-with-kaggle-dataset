# Olist End-to-End Data Analytics Project

## Project Overview

This project demonstrates a complete end-to-end Data Analytics workflow using the Olist Brazilian E-commerce dataset. The objective is to transform raw business data into meaningful insights that support strategic decision-making. The project covers data ingestion, data validation, data modeling, SQL-based analysis, and interactive dashboard reporting.

The solution simulates a real-world analytics environment where transactional and marketing data are processed, transformed, and analyzed to evaluate business performance, customer behavior, seller effectiveness, and revenue trends.

---

## Business Problem

E-commerce businesses generate large volumes of transactional and marketing data. Without proper analytics, it becomes difficult to identify revenue drivers, customer purchasing patterns, marketing effectiveness, and seller performance.

This project aims to:

* Analyze customer purchasing behavior
* Measure seller performance
* Track revenue growth trends
* Evaluate marketing funnel conversion
* Generate actionable business insights
* Build interactive dashboards for stakeholders
